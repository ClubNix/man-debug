# man-debug

aka « Le guide du débogueur galactique ».

Prérequis
---------

- XeLaTeX
- La police "Fira Mono"
- Tommorrow Pygments Theme:
  https://github.com/MozMorris/tomorrow-pygments

Compiler
--------

```bash
latexmk -pdf -shell-escape -xelatex -8bit main.tex
```
